System overview

This upgraded demo includes an interactive map (Leaflet) showing collection coordinates and a QR scanner using the device camera. The admin page provides simple analytics (Chart.js) from demo data.

Demo Features

  • Interactive map with collection markers
  • Camera-based QR scanning (uses jsQR)
  • Admin charts for batch counts and quality metrics

Note: This is still a demo with local JSON data. For production, connect to a backend API and permissioned ledger.
